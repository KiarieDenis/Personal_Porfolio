# Theme
- Grey background (#121212)
- Black cards (#1E1E1E)
- Yellow Highlights (#F1C40F)
- Fonts: 'Space Grotesk' (Headings) & 'Outfit' (Body)

# About the Project
1. Structure: Three core pages (HTML), Flask backend, custom CSS.
2. Hosting: Ready for Render.com or GitHub Pages.
3. Design Principles:
   - Visual Hierarchy: Size & Color guide the eye.
   - Contrast: High contrast for readability.
   - Whitespace: Generous spacing for a premium feel.
   - Animations: Smooth 'rush-in' scroll effects and interactive elements.

# Project Structure
/static
  /css       - style.css (All styles)
  /icons     - SVG icons for Skills Slider
  /images    - Project images & Profile picture
  /certificates - Certificate images
/templates
  - base.html (Header/Footer layout)
  - About_me.html
  - Projects.html
  - Portfolio.html
- main.py    - Flask Application Entry Point

# Features
- Responsive Design: Adapts to mobile and desktop screens.
- Skills Slider: Infinite scrolling loop of tech stack icons.
- Certificate Modal: Lightbox to view certificates in full detail.
- Animations: Slide-in effects triggered by scrolling.
- Interactive Toggle: 'Read My Story' expands to show full biography.

# Setup & Run
1. Install Python.
2. Install Flask: `pip install flask`
3. Run the app: `python main.py`
4. Open browser at: `http://127.0.0.1:5000`
